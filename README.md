## Установка зависимостей

```bash
npm install
```

## Запуск проекта

### Режим разработки

```bash
npm run dev
```

### Сборка проекта

```bash
npm run build
```

### Предпросмотр собранного проекта

```bash
npm run preview
```

## Тестирование

### Запуск тестов

```bash
npm run test
```

### Запуск тестов с UI

```bash
npm run test:ui
```
